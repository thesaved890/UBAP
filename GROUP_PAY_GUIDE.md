# Guide Complet: Paiements de Groupe (Group Pay) - UBAP

## Comment ça fonctionne pour TOUS les utilisateurs

### 📋 Vue d'ensemble

Group Pay permet aux pionniers de créer des cagnottes collectives pour des événements africains typiques: mariages, funérailles, cadeaux d'équipe, fêtes communautaires, etc.

---

## 🎯 Pour l'ORGANISATEUR (Celui qui crée la cagnotte)

### Étape 1: Création de la cagnotte
1. L'organisateur clique sur le bouton **"Group Pay"** dans la barre de navigation
2. Clique sur le bouton **"+"** en haut à droite
3. Remplit le formulaire:
   - **Titre**: Ex: "Mariage de Joseph & Marie"
   - **Description**: Ex: "Collecte pour le cadeau de mariage"
   - **Catégorie**: Mariage, Funérailles, Cadeau, Communauté, Urgence, Autre
   - **Objectif en Pi**: Ex: 0.05 Pi (≈ $15,707)
   - **Monnaie locale**: Ex: CFA (pour recevoir les fonds)
   - **Date limite**: Ex: 25 décembre 2025

### Étape 2: Partage
1. Une fois créée, la cagnotte apparaît dans **"Mes Cagnottes"**
2. L'organisateur partage le lien avec ses contacts via:
   - WhatsApp
   - SMS
   - Réseaux sociaux
   - Email

### Étape 3: Suivi en temps réel
- L'organisateur voit:
  - Le montant collecté en temps réel
  - Le nombre de contributeurs
  - Les noms des personnes qui ont contribué
  - La progression vers l'objectif (barre de progression)

### Étape 4: RÉCEPTION AUTOMATIQUE DES FONDS

**Quand l'objectif est atteint:**

1. **UBAP transfère AUTOMATIQUEMENT** le total collecté dans le compte UBAP de l'organisateur
2. **Conversion automatique** en monnaie locale choisie (ex: CFA, Naira, etc.)
3. Les fonds arrivent dans **"Mon Compte UBAP"**

**Comment l'organisateur utilise les fonds reçus:**

L'organisateur peut utiliser son argent de 3 façons:

#### Option A: Carte Virtuelle UBAP
- Aller dans **"Cards"** (Cartes)
- Recharger sa carte virtuelle avec les fonds reçus
- Utiliser la carte dans **tous les magasins** (supermarchés, boutiques, restaurants)
- Compatible Apple Pay / Google Pay

#### Option B: Virement Bancaire
- Aller dans **"Banks"** (Banques)
- Sélectionner son compte bancaire lié
- Transférer les fonds vers son compte bancaire local
- Retirer à un guichet automatique

#### Option C: Mobile Money
- Aller dans **"Banks"** → **"Mobile Money"**
- Transférer vers Orange Money, MTN Money, etc.
- Retirer chez un agent Mobile Money

### Si objectif NON atteint à la date limite:
- **Remboursement automatique** de tous les contributeurs
- Chaque personne récupère son Pi dans son compte UBAP
- Aucune perte pour personne

---

## 💰 Pour les CONTRIBUTEURS (Ceux qui donnent)

### Étape 1: Accès à la cagnotte
1. Recevoir le lien de l'organisateur
2. Cliquer sur le lien → ouvre UBAP
3. Voir les détails de la cagnotte:
   - Titre et description
   - **NOM DE L'ORGANISATEUR** (visible avec avatar)
   - Objectif et montant déjà collecté
   - Date limite

### Étape 2: Contribution
1. Cliquer sur **"Contribuer"**
2. Entrer le montant en Pi (ex: 0.003 Pi)
3. Voir la conversion en USD et monnaie locale
4. Cliquer sur **"Confirmer la Contribution"**

### Étape 3: Confirmation
- Le Pi est **débité immédiatement** du compte UBAP
- Les fonds vont dans un **compte sécurisé UBAP** (pas directement à l'organisateur)
- Le nom du contributeur apparaît dans la liste
- Notification envoyée à l'organisateur

### Sécurité pour les contributeurs:
✅ Les fonds sont gardés par UBAP (pas par l'organisateur)  
✅ Si objectif atteint → fonds transférés à l'organisateur  
✅ Si objectif NON atteint → remboursement automatique  
✅ Transparence totale: vous voyez qui a contribué

---

## 🔒 Sécurité du Système

### Gestion des fonds par UBAP:

1. **Collecte sécurisée**:
   - Tous les Pi contributés vont dans un compte **escrow UBAP**
   - L'organisateur ne peut PAS toucher les fonds avant l'objectif atteint

2. **Distribution automatique**:
   - UBAP vérifie automatiquement si objectif atteint à la date limite
   - Si OUI: transfert automatique à l'organisateur
   - Si NON: remboursement automatique de tous les contributeurs

3. **Conversion garantie**:
   - Taux GCV fixe: 1 Pi = $314,159
   - Conversion instantanée en monnaie locale
   - Aucun frais caché

4. **Traçabilité**:
   - Historique complet de toutes les contributions
   - Identité vérifiée via Pi Network KYC
   - Pas d'anonymat = pas d'arnaque

## 💡 Cas d'Usage Réels en Afrique

### 1. Mariage (le plus courant)
- **Objectif**: 0.05 - 0.15 Pi ($15,707 - $47,124)
- **Contributeurs**: 20-50 personnes
- **Usage**: Cadeau collectif aux mariés

### 2. Funérailles
- **Objectif**: 0.08 - 0.20 Pi ($25,132 - $62,831)
- **Contributeurs**: 30-100 personnes
- **Usage**: Soutien à la famille endeuillée

### 3. Cadeau d'équipe (bureau)
- **Objectif**: 0.02 - 0.04 Pi ($6,283 - $12,566)
- **Contributeurs**: 10-25 personnes
- **Usage**: Anniversaire, départ à la retraite

### 4. Événement communautaire
- **Objectif**: 0.10 - 0.30 Pi ($31,415 - $94,247)
- **Contributeurs**: 50-200 personnes
- **Usage**: Fête de quartier, église, association

### 5. Urgence médicale
- **Objectif**: Variable
- **Contributeurs**: Variable
- **Usage**: Aide à un membre de la communauté

## ❓ Questions Fréquentes

**Q: L'organisateur peut-il partir avec l'argent avant l'objectif atteint?**  
R: NON. Les fonds restent bloqués dans le compte escrow UBAP jusqu'à ce que l'objectif soit atteint.

**Q: Que se passe-t-il si l'objectif n'est pas atteint?**  
R: Remboursement automatique de tous les contributeurs. Personne ne perd d'argent.

**Q: L'organisateur peut-il modifier l'objectif après création?**  
R: NON. Pour éviter les abus, l'objectif est fixe.

**Q: Les contributeurs peuvent-ils voir qui a donné?**  
R: OUI. Transparence totale pour éviter les fraudes.

**Q: Quels sont les frais?**  
R: 0.5% de frais UBAP prélevés sur le total collecté (seulement si objectif atteint).

**Q: L'organisateur doit-il payer pour créer une cagnotte?**  
R: NON. Création gratuite.

**Q: Peut-on contribuer plusieurs fois?**  
R: OUI. On peut contribuer autant de fois qu'on veut.

**Q: L'organisateur peut-il annuler la cagnotte?**  
R: OUI, mais seulement si aucune contribution n'a été faite. Sinon, il doit attendre la date limite pour remboursement automatique.
